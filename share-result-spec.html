Feature Spec

Share Result

WhoAmI — World Cup 2026 Wordle  ·  Implementation brief for developers

Overview

Goal

After a player finishes the daily WhoAmI challenge, show a modal with their result and let them share it to X (Twitter) or copy to clipboard — without revealing the answer to people who haven't played yet.


Scope

  • Result modal UI (shown on correct guess or all hints exhausted)
  • generateShareText() utility function
  • Copy to clipboard action with toast feedback
  • Share to X via Twitter intent URL
  • Copy link action

Trigger conditions

  • Player guesses correctly → show modal after 1s delay
  • All hints exhausted without correct answer → show modal immediately with "Better luck tomorrow" title
  • Modal also accessible from a "Share" button on the result screen

Dismiss behavior

  • Click outside modal → close
  • Press Escape → close
  • No "X" close button needed (keep UI clean)
  • Cannot reopen same day's modal (use localStorage flag)

Code Reference

generateShareText()

/**
 * Generate spoiler-free share text
 * @param {number} dayIndex       - Puzzle number (e.g. 42)
 * @param {number} score          - Final score 0–100
 * @param {number} hintsUsed      - Number of hints revealed
 * @param {Array}  guessResults   - Array of per-hint results per attempt
 *                                  e.g. [['wrong','wrong','wrong','wrong','wrong'], ['correct','wrong',...]]
 * @returns {string}
 */
function generateShareText(dayIndex, score, hintsUsed, guessResults) {
  const EMOJI = {
    correct: '🟩',
    close:   '🟨',
    wrong:   '⬜',
    unused:  '⬛',
  };

  const HINT_ICONS = '🌍 Nationality · ⚽ Club · 📍 Position · 🔢 Number · 🎂 Age';
  const TOTAL_HINTS = 5;

  // Build emoji grid — one row per guess attempt
  const grid = guessResults
    .map(attempt => {
      const cells = attempt.map(r => EMOJI[r] ?? EMOJI.wrong);
      // Pad unused hint slots
      while (cells.length < TOTAL_HINTS) cells.push(EMOJI.unused);
      return cells.join('');
    })
    .join('\n');

  return [
    `🟩 WhoAmI #${dayIndex} — ${score}/100`,
    grid,
    '',
    HINT_ICONS,
    'wc26wordle.netlify.app/whoami',
  ].join('\n');
}

Share actions

// Copy to clipboard
async function copyResult(shareText) {
  await navigator.clipboard.writeText(shareText);
  showToast('Copied! 📋');
}

// Share to X / Twitter
function shareToX(shareText) {
  const encoded = encodeURIComponent(shareText);
  const url = `https://twitter.com/intent/tweet?text=${encoded}`;
  window.open(url, '_blank', 'noopener');
}

// Toast helper (2 second auto-dismiss)
function showToast(message) {
  const toast = document.createElement('div');
  toast.className = 'toast';
  toast.textContent = message;
  document.body.appendChild(toast);
  setTimeout(() => toast.remove(), 2000);
}

Spoiler Rules

🚫 Never include player name in share text
The share text must be completely spoiler-free. Only emit score, emoji grid, and the game URL.
🔢 Day index must be deterministic
Calculate as Math.floor((today - GAME_START_DATE) / 86400000) so all players share the same puzzle number for the same date.
📋 Clipboard fallback
If navigator.clipboard is unavailable (HTTP or older browser), fall back to document.execCommand('copy') via a temporary textarea.
📱 Mobile share (optional)
If navigator.share is available (mobile browsers), prefer it over clipboard for "Copy Result" — gives native share sheet.

LocalStorage Keys

KeyValuePurpose
whoami-result-YYYY-MM-DD {"score":82,"hintsUsed":3,"guessResults":[...]} Persist result for share modal. Add guessResults field if not already stored.
whoami-played-YYYY-MM-DD "true" Already exists — gate for showing modal on revisit

⚠️ Add guessResults to the result object when saving — required for emoji grid generation.
